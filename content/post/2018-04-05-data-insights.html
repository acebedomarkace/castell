Actionable Analytics Summary:

  1. Regression Analysis - for measuring magnitude and trend
  2. Logistic Regression - measuring probability or likelihood
  3. Clustering - similarities and groupings
  4. Survival Analysis - determining the time to an event
  5. Event Analysis - understanding sequences of complex events
  6. Network Analysis - determining the connections between entities
